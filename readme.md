# Usage
Syntax:

```bash
npx @sturlen/convert AMOUNT FROM TO
```


Example:

```bash
npx @sturlen/convert 1 kilometer miles
0.6
```


Supports SI-prefixes, singular, plural and abbreviations. e.g. `kilometer, kilometers, km`

# Supported units

All SI-units support the following prefixes:
- yotta
- zetta
- exa
- peta
- tera
- giga
- mega
- kilo
- hecto
- deca
- deci
- centi
- milli
- micro
- nano
- pico
- femto
- atto
- yocto
- zepto


## Length
- meter
- inch
- foot
- yard
- mile

## Weight
- gram
- pound
